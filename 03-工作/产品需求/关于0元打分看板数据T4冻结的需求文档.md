# **2. 背景及目标**

**背景：**

目前“0元打分看板”中的各项数据指标会随着时间持续变化。为了在课程结束后的特定时间点固化数据，统一评估标准，业务方提出了数据冻结的需求。经过沟通，从最初讨论的冻结部分关键指标（如上期/上上期辅导类型），最终确认为冻结看板上的所有数据。

**目标:**

统一数据统计口径，确保“0元打分看板”在结课日后第4天（T4）的数据被完整记录且不再变动，为后续的复盘和分析提供稳定、一致的数据基础。

# **5. 功能描述**

本次需求不涉及前端页面或交互的改动，是对后端数据逻辑进行修改。

- **数据冻结点：** 以在T+4日（结课日）的24点进行数据冻结。
    
- **冻结范围：** “0元打分看板”上展示的所有数据指标。
    
- **用户影响：** 用户（看板使用者）在T4之后查看看板时，看到的数据将是恒定不变的T4数据快照。
